# 🚀 Vyom Rockets

A modern, high-performance, and fully responsive landing page built to showcase next-generation spacecraft. This project highlights fluid responsive layouts, clean semantic HTML structure, and semantic interactive elements.

🔗 **Live Deployment:** [View Vyom Rockets Live on Render](https://vyom-rockets.onrender.com)

---

## ✨ Features & Highlights

* **Tailwind CSS v4 Integration:** Migrated fully to the latest CSS-first configuration, utilizing `@theme` directives and native CSS variables for superior performance and instantaneous compilation.
* **Custom Keyframe Animations:** Features a customized drop-down bounce menu execution using optimized scaling keyframes (`open-menu`).
* **Interactive Hamburger Menu:** Dynamic DOM manipulation using JavaScript to smoothly morph the mobile navigation icon into an 'X' using native translation and rotational properties.
* **Responsive Architecture:** Engineered using a mobile-first approach, scaling seamlessly from small smartphone screens to wide-screen monitor orientations (`widescreen` & `tallscreen` aspect ratios).
* **Dark Mode Compatibility Ready:** Configured to match layout standards using baseline dark utilities.

---

## 🛠️ Built With

* **HTML5** - Semantic layout structure
* **Tailwind CSS v4** - Utility-first styling framework
* **JavaScript (ES6)** - Interactive DOM event handling & responsive menu toggles
* **Render** - Production build deployment platform
